# Pokémon Bank

Proyecto desarrollado como parte de la asignatura Desarrollo de Aplicaciones Web (DAW).

## Descripción

Pokémon Bank es una interfaz web que simula el funcionamiento de un cajero automático (ATM), permitiendo al usuario realizar operaciones básicas como depósito, retiro, consulta de saldo y pago de servicios.

Este proyecto corresponde a la fase de desarrollo Front-End, utilizando tecnologías del lado del cliente.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- Chart.js

---

## Funcionalidades

- Login mediante PIN (simulación)
- Menú principal de operaciones
- Depósito de dinero
- Retiro de dinero
- Consulta de saldo
- Pago de servicios
- Historial de transacciones
- Gráfica de operaciones
